# CRICKSTATS – Cricket Player Performance Tracking System
Project Overview

CRICKSTATS is a cloud-based cricket player performance tracking application developed using Salesforce. The system allows coaches, managers, and analysts to store, manage, and analyze player statistics for each match. It centralizes performance data and provides automated analytics through reports and dashboards.

The project follows the Software Development Life Cycle (SDLC) and Agile methodology to design and implement a scalable performance tracking platform. 


# Team Activities

The project was completed collaboratively by the team members by performing the following activities:

# 1. Project Planning & Ideation

* Identified the problem in traditional cricket performance tracking.
* Analyzed user requirements for coaches, managers, and analysts.
* Designed solution architecture for the CRICKSTATS system.

# 2. Salesforce Environment Setup

* Created Salesforce Developer accounts.
* Configured the development environment.
* Set up the workspace for application development.

# 3. Data Modeling

* Created custom objects:
* Player
* Match Performance
* Implemented relationships between objects using Master-Detail and Lookup relationships.

# 4. User Interface Development

* Created custom tabs for navigation.
* Developed the CRICKSTATS Lightning Application.
* Designed page layouts and user-friendly forms.

# 5. Field Configuration

Added custom fields such as:

* Player Name
* Role
* Batting Style
* Bowling Style
* Runs Scored
* Wickets Taken

Implemented formula fields and picklists for structured data entry.

# 6. Automation Implementation

Developed Salesforce Flows to automate data processing.

Configured roll-up summary fields to calculate statistics such as:

* Total Runs
* Matches Played
* Total Wickets
* Player Average Score.

# 7. Reporting & Analytics

* Created reports and dashboards to visualize player performance.
* Implemented analytics for match-by-match statistics.

8. Testing

* Performed functional testing to verify object creation, flows, and automation.
* Evaluated system performance for report generation and dashboard loading.
* Technologies Used
* Salesforce Lightning
* Salesforce Objects & Fields
* Salesforce Flow Automation
* Reports & Dashboards

* Cloud Database (Salesforce Platform)
